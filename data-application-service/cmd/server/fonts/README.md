# 字体文件说明

此目录包含 PDF 导出所需的中文字体文件：

- `simfang.ttf` - 仿宋字体（用于正文）
- `simhei.ttf` - 黑体字体（用于标题和粗体）

这些字体文件需要从 data-view 工程复制过来，确保中文字符能正确显示。

## 复制命令

```bash
# 从 data-view 工程复制字体文件
cp D:\go_workplace\data-view\cmd\server\fonts\simfang.ttf cmd\server\fonts\
cp D:\go_workplace\data-view\cmd\server\fonts\simhei.ttf cmd\server\fonts\
```
